### Tech Stack

- node v14
- npm v6
- create-react-app

### Default Branch
- master


## Installation Guides

### Manual/local build

```bash
# provide environment variables in a .env file
$ cp .env.example .env.local

# install node dependencies
$ npm i

# run app locally
$ npm start
or
# build app for production
$ npm run build
```
